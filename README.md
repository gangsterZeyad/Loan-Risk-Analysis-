# Loan Risk Analysis 📊

## 📝 Overview  
مشروع تحليل مخاطر القروض باستخدام Power BI و Python وملفات CSV، و MySQL  يهدف لتحديد العوامل المؤثرة على القروض وتقديم Insights.
## 📂 Project Structure  
- **📁 Data** → ملفات CSV الخاصة بالقروض  
- **📊 Power BI** → تقارير تحليلية عن القروض  
- **📜 EDA (Python)** → تحليل البيانات باستخدام Python  

## 🚀 Technologies Used  
- Python 🐍  
- Power BI 📊  
- Pandas & NumPy 📈  
- Matplotlib & Seaborn 🎨  

## 📌 How to Use  
1. افتح ملف Power BI لاستعراض التحليل.  
2. شغّل كود Python لاستكشاف البيانات.
